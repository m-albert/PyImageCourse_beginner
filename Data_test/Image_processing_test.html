Test for Workshop in image processing with Python#

Write your answers as code in one or multiple cells below each question. Check that your code works by executing each cell. If your code sovles the problem you get all points. If your code does not work or you are not confident in your answer, you can add comments describing your code and get up to half the points of the question.

For some questions, basic data are provided to you through an initial cell that you have to execute to be able to use these data.

To avoid that anyone gets completely stuck because of wrongly imported modules, you can import all the necessary ones by executing the cell below. Be aware of the abbreviations used (np, pd, plt).

import numpy as np
import pandas as pd
import skimage
import skimage.io
import skimage.filters
import skimage.measure
import matplotlib.pyplot as plt
