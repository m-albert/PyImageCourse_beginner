2. Images as arrays exercises#

  1. Import Numpy

  1. Create a 6 rows, 4 columns array of 1s called my_array. Using the shape parameter make sure the size is correct.

  1. Modify the element at row = 3, column = 1 to be 50

  1. Modify the array by multiplying all its values by 3

  1. Using Google, find the Numpy function to take the square root and create a new array called my_array_root by applying it to your current my_array.

  1. Create a new array called my_array_sum by multiplying my_array and my_array_root

  1. Find the maximum value of your array by using either a methods or a function

  1. Create a boolean array where only places in my_array_sum which are larger than 100 are True.

  1. Recover all the values in my_array_sum which are true in the boolean array using logical indexing. What is the shape of the array containing these values?
